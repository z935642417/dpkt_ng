## 2. Architecture: TopologyBuilder

### Flow

```
PCAP → Ethernet → IP
  ├── proto=89 (OSPF) → ospf.py → extract_ospf()
  ├── proto=88 (EIGRP) → eigrp.py → extract_eigrp()
  ├── proto=17 (UDP):520 → rip.py → extract_rip()
  ├── proto=6 (TCP):179 → stream.py reassemble → bgp.py → extract_bgp()
  └── LLC dsap=0xfe → isis.py → extract_isis()
```

### TCP Stream Integration for BGP

```python
class TopologyBuilder:
    def __init__(self):
        self._bgp_stream = StreamReassembler()  # TCP reassembly for BGP
    
    def process_packet(self, ip, eth=None):
        if isinstance(ip.data, TCP) and (ip.data.sport == 179 or ip.data.dport == 179):
            self._bgp_stream.feed(ip, ip.data)  # feed to TCP reassembler
        elif isinstance(ip.data, OSPF):
            self._extract_ospf(ip, ip.data)
        # ... other protocols
```

## 3. Protocol Extractors (5 methods)

| Method | Input | Extraction |
|--------|-------|-----------|
| `_extract_ospf(ip, ospf)` | LSU → LSAs | Router-LSA links, Network-LSA routers, External prefixes |
| `_extract_isis(isis)` | LSP → TLVs | Extended IS Reach(22), IP Internal(128), IP External(130) |
| `_extract_bgp(ip, bgp)` | UPDATE → NLRI+attributes | Prefix + AS_PATH + NEXTHOP + ORIGIN |
| `_extract_rip(ip, rip)` | Response → RIPEntry | Prefix + subnet_mask + nexthop + metric |
| `_extract_eigrp(ip, eigrp)` | Update → Route TLVs | Internal(0x0102)/External(0x0103) → Prefix |
